简介

项目基于 parcel 构建, 使用 bootstrap 和 jquery. 具体引入方式与页面基础结构查看/page/movie.html的代码注释,项目启动后编辑代码会自动加载无需刷新页面

技术文档

库名 作用 文档地址
parcel 构建工具提供开发服务器和热加载 https://www.parceljs.cn/getting_started.html
bootstrap 提供样式 https://v4.bootcss.com/docs/getting-started/introduction/
jquery 这个库封装大部分原生js操作用起来比较方便 https://www.w3school.com.cn/jquery/index.asp
You Don't Need jQuery 这个是个技术文档介绍了大部分常见js操作页面的方式 https://github.com/nefe/You-Dont-Need-jQuery/blob/master/README.zh-CN.md
douban-api-docs 豆瓣api https://douban-api-docs.zce.me/

项目结构

page 文件夹存放页面,文件夹下的 movie.html 页面, movie.js 是页面配套的js文件

index.html是首页,index.js是首页配套js文件

style.css是自定义样式

components是组件库

其他的文件暂时不用了解
